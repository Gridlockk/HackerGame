#  Hacker Console Game

Консольная игра в жанре киберпанк‑квеста. Вы — оператор, которому поручено восстановить украденные данные и проследить их источник. Исследуйте серверы, находите зашифрованные файлы, подбирайте ключи и взламывайте системы.

##  Требования

- [.NET SDK 8.0](https://dotnet.microsoft.com/download/dotnet/8.0) или новее (рекомендуется 8.0 LTS).  
  Проект использует только библиотеку `Spectre.Console` 0.57.2, которая совместима с .NET 8.0+.

##  Запуск

1. Клонируйте репозиторий:
   git clone <url-репозитория>
   cd HackerConsole
2. Выполните сборку и запуск:
   dotnet run

При первом запуске автоматически создастся файл сохранения save.json. Игра загрузит квест из quest/quest.json



##  Команды
Команда	Описание
help	        Показать список доступных команд
ls	            Вывести содержимое текущей директории (файлы и папки)
cd <папка>	    Перейти в указанную папку; cd .. — на уровень выше
cat <файл>	    Прочитать содержимое файла (если он не зашифрован)
decrypt <файл>	Расшифровать зашифрованный файл (потребуется ключ)
hack <адрес>	Попытаться взломать сервер (ввести пароль)
scan	        Просканировать сеть: показать доступные серверы и подсказку
connect <адрес>	Подключиться к указанному серверу
status	        Показать текущий прогресс: сервер, директория, уровень, очки, ключи
clear	        Очистить экран
exit	        Выйти из игры
